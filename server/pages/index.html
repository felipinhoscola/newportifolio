    To-Do List

    Projeto To-Do List com Node.js, Express, MongoDB e Mongoose

    Meu projeto To-Do List é o resultado da união entre tecnologias de ponta. Utilizando Node.js e Express, criei uma plataforma ágil e responsiva para gerenciar tarefas. O armazenamento seguro é garantido pelo MongoDB, implementado com o auxílio do Mongoose. Convido você a experimentar a eficiência e praticidade desta aplicação, onde a organização de afazeres se encontra com a inovação tecnológica.
